Nexora Connect
Alvorada Motocicletas · v1.0 · 2026
Plataforma de captação e relacionamento comercial desenvolvida para centralizar a gestão de representantes, clientes, vendas e retornos em um único ambiente profissional.
Funcionalidades
✓Captação de clientes via CNPJ com 4 fontes simultâneas
✓Pesquisa via IA com busca no Google
✓Ficha completa de clientes com histórico de contatos
✓Controle de retornos e alertas de inatividade
✓Gestão de vendas e parcelamentos
✓Biblioteca de modelos de comunicação para WhatsApp
✓Dashboard comercial com ranking e metas por representante
✓Relatórios Excel profissionais com gráficos
✓Auditoria completa de todas as ações
✓Controle de acesso por perfil: Representante, Gerente e ADM
✓Integração com Google Sheets como banco de dados
Informações técnicas
Tecnologia: HTML · CSS · JavaScript · Google Apps Script
Deploy: GitHub Pages — 100% gratuito
Banco de dados: Google Sheets + localStorage
